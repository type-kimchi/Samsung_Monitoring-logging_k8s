# Samsung_Monitoring-logging_k8s
Samsung Electronics DS project with Kyndrl, Samsung Electronics


# DS Cloud Container 서비스 내재화

## 목적

필요한 S/W를 플랫폼에 자유롭게 반영
별도의 비용 부담 없이 플랫폼 배포/확대 가능
사용목적 및 운영 수준에 따라 다양한 형태로 제공
